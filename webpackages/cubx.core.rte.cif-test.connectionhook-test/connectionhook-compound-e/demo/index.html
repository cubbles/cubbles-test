Demo page for compound component connectionhook-compound-e

This compound component demostrate the global hookFunction between a member and self. The compound hast one member. : connectionhook-elementary-a

Connections

  • connection between "member-a"/slot "a" and slot "aa" has a connection global hookFunction "cubx.hookFunctions.multiply10." (The function multiply payload with 10. The payload is a number.)
  • connection between "member-a"/slot "b" and slot "bb" has a connection global hookFunction "cubx.hookFunctions.greeting." (The function include "Greeting" in first position to payload. The payload is a string.)
  • connection between "member-a"/slot "c" and slot "cc" has a connection global hookFunction "cubx.hookFunctions.fullName." (concat object attributes name and firstname to a object attribte name. The paylod is an object)

The global functions are in the utility artifact transform-function defined.
