Automatic Conflict Resolution (ACR)Test

Resolve version conflict

Scenario: <compund-a> and <compund-b> both are using <artifact-a> from same webpackge but with different versions. In the DOM tree <compund-b> is the first item and <compund-a> the second item so in dependency tree <compound-b> comes before <compund-a> (level order traversal). Thus ACR using shortest path lookup will resolve this version conflict choosing <artifact-a@1.1> over <artifact-a@1.0>.

Expected resolution:

<artifact-c@1.0>

<artifact-a@1.1>

<artifact-a@1.1>

<artifact-b@1.0>
