Demo page for the compound component copy-value-test-compound

  • contains 2 elementary components: copy-value-test-a and copy-value-test-b
  • all output slots (a,b,c,d) from copy-value-test-a is connected to the input slots (a,b,c,d) of copy-value-test-b
  • all input slots from this compound component (aa, bb, cc, dd) is connected to the input slots (a,b,c,d) of copy-value-test-a
  • all output slots (a,b,c,d) of copy-value-test-a is connected to the output slots if this compound component (aa, bb, cc, dd)

For the following connections is the "copyValue" attribute false

  • copy-value-test-obj-a/c -> copy-value-test-obj-b/c
  • copy-value-test-obj-a/c -> copy-value-test-compound-obj/cc
  • copy-value-test-compound-obj/cc -> copy-value-test-obj-a/c
for the following connections is the "copyValue" attribute explicitly true
  • copy-value-test-obj-a/b -> copy-value-test-obj-b/b
  • copy-value-test-obj-a/b -> copy-value-test-compound-obj/bb
  • copy-value-test-compound-obj/bb -> copy-value-test-obj-a/b
for all other connections is the "copyValue" attribute implicitly true (copyValue attribute missed)
