Demo-app for add and remove dynamic connections

try the show case:

  1. change the message (leave the input box) - nothing happend, the container-element and the transportable-element are not connected yet.
  2. drag transportable-element and drop in the container drop zone
  3. change the message (leave the input box) - transportable-element recived and show the message, the container-element and the transportable-element are connected
  4. drag transportable-element and drop in dynamic-connection-compound (the place with grey background)
  5. change the message (leave the input box) - transportable-element not recived and show the new message, the container-element and the transportable-element are not connected anymore.
