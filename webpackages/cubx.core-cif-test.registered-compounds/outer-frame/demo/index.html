Demo site for check registration of custom html tags.

Once the tag registered, you can use the cubbles api functions. This will be done with the on change handler function of the message input field.


        function changeHandler(elem,event){
            console.log('elem',elem);
            console.log('event',event);
            var outerFrame = document.querySelector('outer-frame');
            outerFrame.setInputSlot('message', {
                slot: 'message',
                payload: elem.value
            })
        }
